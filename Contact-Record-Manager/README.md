# Contact Record Manager
A simple Contact Record Management program written in C language through which we can
add, view, update and delete student record using struct and file handling

## Feature 🛠️
- Add new contact
- View all contact
- Search contact by phone number or name
- Edit contact information
- Delete contact
- Phone and Name are both unique for each contact

## Technologies Used 🖥️
- C Programming Language
- GCC Compiler

## Key Concepts Used 🔑
- File Handling
- Structures in C
- Input validation
- Modular functions

# Installation ⚙️

1. Clone the repository
git clone https://github.com/kirtanpatidar-dev/C-Projects.git

2. Go to the project folder
cd Contact-record-manager

3. Compile the program
gcc main.c -o contact_manager

4. Run thee program
./contact_manager

## Usage 🕹️
Run the program and choose the options from the menu to manage contact data.
