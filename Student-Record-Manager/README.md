# Student Record Manager 🧑‍🎓
A simple Student Record Management System written in C language through which we can
add, view, update and delete student record using struct and struct handling

## Feature 🛠️
- Add new students
- View all students
- Search students by roll number or name
- Update student information
- Delete student data

## Technologies Used 🖥️
- C Programming Language
- GCC Compiler

## Key Concepts Used 🔑
- Structures in C
- Input validation
- Modular functions
- Dynamic memory allocation

## Installation ⚙️

1. Clone the repository
git clone https://github.com/kirtanpatidar-dev/C-Projects.git

2. Go to the project folder
cd Student-record-manager

3. Compile the program
gcc main.c -o student_manager

4. Run thee program
./student_manager

## Usage 🕹️
Run the program and choose the options from the menu to manage student data.
